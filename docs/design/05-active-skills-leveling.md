# 绝技逐个升级曲线（Lv1~5）· 明朝抗倭

> 配合 03（技能清单）与 04（数值定档）。本表给**每个绝技独立的 5 级强化曲线**。
> 伤害以「×攻击」系数表示（吃 04 文档的乘区结构）。CD 为基础冷却（吃冷却缩减）。
> 升级方式：三选一中重复出现同名绝技即 +1 级；满 Lv5 后该卡转为「精通」词条（小幅全局加成）。
> 绝技携带上限：6 个绝技槽（与武器/被动共享构筑空间，详见后续平衡文档）。

图例：`伤害` = 单次命中系数 ×攻击；`数量` = 弹体/目标/召唤数；`异常` = 附加异常层数。

## 平衡备注
- 所有「×攻击」系数为**基准**，将随关卡难度做整体缩放校准。
- 觉醒类同一时间只生效 1 个（避免大招互相覆盖），其余觉醒进入共享短冷却。
- 部署/召唤类受「连营/死士」词条强化，受「绝技数量上限」约束。
- 满 Lv5 后重复出现 → 转「精通」词条（该绝技伤害 +小幅 / 或全局对应属性 +小幅）。

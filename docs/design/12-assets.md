# 素材生产计划（生产级美术资源）· 明朝抗倭

> 用户要求:**MVP 阶段除开发逻辑外,需一并产出生产级素材**——场景、人物、技能特效、状态、图标、NPC/敌人。本文定义美术方向、资源清单、生产管线与交付节奏。素材在 M1~M8 随对应系统逐步产出,先占位跑通管线、再替换为生产级。

---

## 一、美术方向（统一风格,确保协调）

- **题材**:明朝中后期·东南沿海抗倭。江南水乡 + 卫所军营基调。
- **视角**:2D 俯视/斜俯视(top-down),贴合割草大量同屏单位。
- **风格基调**:中式水墨/工笔上色质感的 2D 卡通,描边清晰、剪影易辨识(割草必须一眼区分敌我/可拾取/危险)。色板偏青绿水乡 + 朱红/玄黑(明军)对比倭寇的灰褐。
- **统一规范**(写入 `AssetRegistry` 约定):
  - 角色/敌人:统一像素密度与基准尺寸(如角色 64×64 基准),朝向 4 向或 8 向(MVP 可先单向+水平翻转)。
  - 帧动画:idle / run / attack / hit / death 基础集;绝技释放另配专属帧。
  - 特效:序列帧 or 粒子,统一锚点与混合模式。
  - 图标:统一尺寸(如 64×64)、统一描边/底框,绝技/词条/增益/状态四类风格一致。

## 二、MVP 资源清单（对齐 08:1 角色 / 1 图 / 10 绝技 / 6 异常）

### 场景(江南水乡轻量版)
- 地表 tile/背景图(水乡:石板、水道、屋舍、芦苇),可平铺;竞技场边界视觉。
- 环境装饰物(可选静态:石桥、船、灯笼)。
- 关卡氛围(白天基调;Boss 战可切色调)。

### 人物(玩家)
- **刀牌手**:idle/run/attack/hit/death 帧 + 戚家刀挥砍动作。
- 受击闪白、死亡反馈复用 PresentationSystem。

### 敌人 / NPC
- 倭寇基础兵(散兵/刀兵):idle/run/attack/hit/death。
- 远程兵(铁炮/弓):含射击帧。
- 精英(官位词缀,带光环视觉)。
- mini-boss + 关末家族 Boss:更大体型 + 专属动作/血条。
- (可选)友方 NPC/据点视觉。

### 技能特效(10 绝技,覆盖全类型)
- 每绝技:释放特效 + 飞行/轨迹视觉 + 命中特效(对齐 09 二之三 PresentationSpec、二之四 MotionSpec)。
- 类型覆盖:发射型(弹幕/链式/投掷)、区域型(光环/毒区)、单位型(召唤/部署)。

### 状态(6 异常视觉)
- 创伤/焚烧/瘴毒/雷殛/震慑/破甲:敌人身上的覆盖特效 + 状态图标。色彩区分(火橙/毒绿/雷紫…),色盲友好备选(C 档无障碍)。

### 图标
- 绝技图标 ×10、词条/增益图标(MVP 子集)、状态图标 ×6、UI 图标(血/经验/货币/计时/重抽)。

### UI / HUD 素材
- 血条/经验条/绝技 CD 框、三选一卡面底框、Boss 血条、结算页底图(占位)。

### 音频(配合 A4)
- BGM(战斗/Boss)、关键 SFX(挥砍/命中/暴击/拾取/升级/异常爆发/Boss 出场)。

## 三、生产管线

1. **占位先行**:M0~M1 用简单色块/几何体跑通管线与 `AssetRegistry` key 体系,确保"换素材=零逻辑改动"。
2. **生产级产出**:用 AI 生成(`generate_image`)按统一风格批量产出角色/敌人/图标/特效底图,必要时切帧/打图集(texture atlas)。保持同一 prompt 风格模板确保协调。
3. **整合**:资源进 `public/assets/`(或 `src/assets/`),在 `AssetRegistry` 注册 key→图集/帧/粒子/音频;`PreloadScene` 预加载。
4. **优化**:打 spritesheet/atlas 合并 draw call;压缩;按需懒加载。
5. **交付验收**:每里程碑在线链接上由用户手动验收美术观感,反馈后迭代。

> 注:AI 生成素材风格一致性、商用授权与 Steam 上架合规需留意;最终发行可能需美术二次精修或替换,但**逻辑/接口不受影响**(素材即数据,经 key 引用)。这一点在进入发行阶段前与用户确认。

## 四、与里程碑映射

| 里程碑 | 配套素材 |
|---|---|
| M0 | 占位资源 + AssetRegistry/Preload 管线 |
| M1 | 刀牌手 + 基础倭寇兵 + 戚家刀挥砍/命中特效 + 水乡地表 |
| M2~M3 | 经验球/拾取/升级 UI、三选一卡面、图标首批 |
| M4 | 10 绝技释放/轨迹/命中特效 + 绝技图标 |
| M5 | 6 异常覆盖特效 + 状态图标 + 爆发特效 |
| M6 | 远程/精英/mini-boss/关末 Boss + Boss 血条 + 关卡氛围 |
| M7~M8 | HUD 完整、结算页、武器进化视觉、音频(BGM/SFX)、打磨统一风格 |

> 原则:**素材是数据(经 key 引用),与逻辑解耦**;先占位保证可玩验证,再逐里程碑替换为生产级,用户在线手动验收。

# 设计决策日志 (Decisions Log)

> 本文件记录与 Alex 在规划阶段达成的所有共识，定稿后整理为正式设计文档。

## 已确认
- **技术栈**: TypeScript + Phaser 3
- **视角**: 2D 俯视角 (top-down)
- **攻击模式**: A2 — 自动为主 + 1~2 个手动主动技能键
- **伤害成长来源**: B3 — 等级 / 武器进化 / 局外 meta 三者并重
- **核心属性集**: C2 完整版（攻击力、攻速、技能范围、暴击率、暴伤、移速、拾取范围、冷却缩减、持续时间、投射物数量、幸运、最大生命、生命回复、护甲、经验加成）

## 待确认
- **题材风格**: 已定 — 像素肉鸽 + 明朝抗倭背景
  - 主角：明朝中原人，多种职业（兵种），保家卫国
  - 敌人：倭寇，多种职业/官位/家族身份
  - 美术：像素风
### 技能体系（已定主框架）
- 四层结构：主武器(自动) / 被动(兵书军备) / 武器进化 / 主动技能(手动键)
- 武器获取：**丙 混合** — 职业给「专属起手武器 + 专属主动技能」，其余武器槽局内自由拾取
- 职业差异化：**中等偏深** — 专属主动 + 专属进化 + 被动倾向
- 协同系统：**做「鸳鸯阵」武器协同**（本作特色）
- 鸳鸯阵协同：4 类兵种位（盾卫/近战/杂兵/火器），阶梯激活 2位→3位→4位
  - 效果方向：**更质变** — 4 位齐全召唤 NPC 战友一起割草（同时保留低阶 buff）
- 初始职业（6 个，已定）：刀牌手 / 长枪兵 / 鸟铳手 / 狼筅兵 / 军医 / 火器营
  - 每职业 = 专属起手武器 + 专属主动技能 + 被动倾向
- 武器进化清单（已定）：戚家刀法·破阵 / 拒马连环枪 / 铁狼筅·荆棘 / 盾山阵 / 三段连发铳 / 红夷大炮 / 一窝蜂 / 连珠箭 / 悬壶济世
- 主动技能上限：**2 个**（手动键）
- **【修订】主动技能 + 觉醒技 = 局外成长（meta loadout）**：在卫所解锁/升级，开局前装配；不再进入局内三选一池
  - 局内三选一池只含：词条 / 绝技(自动触发) / 联动(增益) / 流派核心
  - 觉醒技从「绝技B类」移出，归入局外装配（每局可带 1 个觉醒）
- 稀有度：**白→绿→蓝→紫→金**，影响出现概率与强度，配合「幸运」属性

### 地图玩法（已定主框架）
- 模式：**战役关卡制（主打）+ 无尽模式（解锁）**；每关 15~20 分钟，时间推进刷怪强度，末尾倭寇头目 Boss
- 关卡间在「军营/卫所」做局外成长
- 地形互动：**先轻量（固定地形+资源点）后期加重（动态地形/机关/环境危险）**
- 地图数量：**起步 3 张精做**（5 个候选：滩涂/海岛巢穴/江南水乡/台州守城/山地隘口）
- 资源点：粮仓(回血) / 军械库(随机武器) / 烽火台(援军/增益)
- 动态事件：倭寇增援 / 百姓护送 / 夺旗破阵
- 精做 3 张（已定）：**江南水乡(教学) → 东南沿海·滩涂(中期,潮汐) → 倭寇巢穴·海岛(终章,火攻+Boss)**
  - 守城战/山地隘口留作后续扩展

### 核心循环（已定）
- 局内：击杀掉军功值(经验)→升级三选一(武器/被动/进化)；拾取银两(临时增益)/粮草(回血)/军械库(随机武器)
- 局外（卫所 meta）：军功→解锁职业/武器/地图；白银→永久属性强化树；兵书残卷→提高高稀有度出现率
- meta 侧重：**平衡（解锁 + 适度数值）**
- 死亡惩罚：**经典 — 一局清空，只留 meta 货币**
- 难度阶梯：普通/困难/地狱，高难更好产出

### 战斗细节 + 敌人体系（已定）
- 伤害公式：**乘区 + 护甲减法 + 暴击**
  - `伤害 = (武器基础 × (1+攻击%)) × 暴击倍率 × 阵法增伤 − 护甲`；暴击默认 ×2；部分武器破甲
  - DPS = 基础 × 攻速 × 投射物数 × 命中数；所有词条换算为乘区避免爆炸
- 刷怪：时间轴推进密度/新兵种，每 ~5 分钟一次倭寇增援潮；精英带词缀；关末 Boss
- 倭寇三层体系：
  - 职业(行为)：足轻/浪人/弓兵/铁炮足轻/忍者/僧兵
  - 官位(精英强度+光环)：组头/侍大将/军师
  - 家族/势力(Boss)：每图终章家族头目，专属机制（真田水军/柳生剑豪/大友铁炮头 等）
- **轻度属性克制**：火器克披甲、近战克散兵 等
- Boss：3 关 3 家族 Boss + **关卡中期 mini-boss**

### 角色 / UI / 美术（已定）
- 角色解锁：开局送刀牌手，其余 5 职业用军功解锁；每职业 3 级熟练度；隐藏角色戚继光
- UI：HUD(血/护甲/计时/槽位/主动CD) + 升级三选一卷轴卡牌(稀有度色边)；中式命名(卫所/校场/军械库/兵书阁)；衬线/书法体
- 操作：WASD/方向键，自动攻击，主动技能 J/K 或 空格/右键，手柄支持
- 美术：像素风 32×32，俯视；明军青砖朱红 vs 倭寇暗甲；国风战斗乐(唢呐/笛/鼓)
- 资源策略：原型先占位美术，确认手感后替换正式美术

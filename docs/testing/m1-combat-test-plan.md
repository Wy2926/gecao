# M1 战斗核心 — 运行时测试计划 (PR #4)

环境：本地 `vite preview` → http://localhost:4173 （dist 已含透明精灵，md5 与 served 一致）

## 测试用例（对抗式，每步都能区分"工作/损坏"）

### T1 — 透明精灵渲染（修复白底方块）

- 操作：加载页面，观察玩家精灵。
- 通过：刀牌手/倭寇为透明背景，无白色方块包围。
- 失败（旧 bug）：精灵周围有白色旋转方块。

### T2 — 自动攻击 + 击杀计数

- 操作：用 WASD 移动靠近来袭倭寇，停留观察戚家刀扇形横扫。
- 通过：扇形特效出现 → 倭寇受击闪白 → 被击退 → 消失；右上角"击杀"从 0 递增（出现 ≥3）。
- 失败：倭寇不掉血/不消失，或"击杀"恒为 0。

### T3 — 接触掉血 + 相机震屏

- 操作：故意贴近倭寇让其接触玩家。
- 通过：左上角红色血条明显变短；屏幕轻微震动。
- 失败：血条不变 / 无震屏。

### T4 — 相机跟随

- 操作：持续向一个方向移动数秒。
- 通过：水乡地表相对滚动，玩家保持在视口中心死区内。
- 失败：玩家移出视口或相机不动。

### T5 — 死亡 + 重开

- 操作：停止移动，任由多个倭寇围殴直到血条空。
- 通过：出现"阵亡"覆盖层与"按 R 重新开始"；按 R 后血条满、击杀归零、重新开始。
- 失败：无覆盖层 / R 无效。
